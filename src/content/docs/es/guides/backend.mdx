## ¿Qué es un servicio backend? \{#what-is-a-backend-service\}

Un servicio backend es un sistema basado en la nube que te ayuda a construir y administrar la infraestructura de tu backend. Proporciona un conjunto de herramientas y servicios para administrar bases de datos, autenticación de usuarios y otras funcionalidades en el lado del servidor. Esto te permite enfocarte en construir tus aplicaciones sin tener que preocuparte por administrar la infraestructura subyacente.

## ¿Por qué usaría un servicio backend? \{#why-would-i-use-a-backend-service\}

Podrías considerar un servicio backend si tu proyecto tiene necesidades complejas en el lado del servidor, por ejemplo:
- registro y autenticación de usuarios
- almacenamiento persistente de datos
- almacenamiento de activos cargados por los usuarios
- generación de API
- comunicación en tiempo real
- monitoreo de aplicación
