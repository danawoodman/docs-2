> **GetStaticPathsRequired**: `getStaticPaths()` function is required for dynamic routes. Make sure that you `export` a `getStaticPaths` function from your dynamic route.

## Qu'est-ce qui a mal tourné ? \{#what-went-wrong\}
En [Mode Statique](/fr/guides/routing/#static-ssg-mode), toutes les routes doivent être déterminées au moment de la construction. Ainsi, les routes dynamiques doivent `exporter` une fonction `getStaticPaths` retournant les différents chemins à générer.
