> **PrerenderClientAddressNotAvailable**: `Astro.clientAddress` cannot be used inside prerendered routes

## Qu'est-ce qui a mal tourné ? \{#what-went-wrong\}
La propriété `Astro.clientAddress` ne peut pas être utilisée dans les routes pré-rendues.
